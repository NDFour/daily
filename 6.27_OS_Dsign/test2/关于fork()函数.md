# 关于fork()函数



- 关于fork()函数:

  fork（）函数调用时，在调用的地方立马创建一个子进程，子进程同时立马开始执行代码（全部代码）。对于以下代码：

  ```c
   12 int main()
   13 {
   14         int x;
   15         printf("1111");
   16         x=fork();
   17         printf("222");
   18         printf("x=%d\n",x);
   19         printf("333\n");
   20         int i=0;
   21         int b=9;
   22         while(i<9)
   23         {
   24             i++;
   25             b=i;
   26         }
   27         printf("finally\n");
   28 }
  ​```
  ```

  当**22**行为`while(i<999999)`时，运行结果为:

  ```c
  1111222x=22093
  333
  1111222x=0
  333
  finally
  finally 
  ```

  当**22**行为`while(i<9)`时，运行结果为：

  ```c
  1111222x=22093
  333
  finally
  1111222x=0
  333
  finally 
  ```

  由此可见，执行*父进程*执行`fork()`函数之后，当即产生一个*子进程*，并且同时子进程开始运行(相同的代码)。
